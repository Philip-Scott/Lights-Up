## Installation

### Dependencies
These dependencies must be present before building
 - `meson (>=0.40)`
 - `valac (>=0.16)`
 - `debhelper (>= 9)`
 - `libgranite-dev`
 - `libgtk-3-dev`
 - `libsoup`
 - `libjson-glib-dev`


 ### Building

```
meson build && cd build
meson configure -Dprefix=/usr
ninja
sudo ninja install
com.github.philip-scott.lights-up
```
